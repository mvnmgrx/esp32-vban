### Configure the project
```
idf.py menuconfig
```

### Build, flash, and run
Build the project and flash it to the board:

```
idf.py build
idf.py -p PORT flash
```

(Replace PORT with the name of the serial port to use.)

The device sends periodic UART output to show debugging counters.
